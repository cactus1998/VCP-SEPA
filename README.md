# AG QUANT SEPA SCREENER & VCP DIAGNOSTICS
### 專業級股票趨勢篩選與波動收縮型態（VCP）診斷系統

本專案是一個針對美股市場開發的**高效率、無外部相依性（Zero Dependencies）**量化技術面篩選與個股型態診斷系統。
本系統基於超級績效大師**馬克．米奈爾維尼（Mark Minervini）**的 **SEPA（特定進場點分析）** 與 **VCP（波動收縮型態）** 核心交易邏輯實作。系統會自動抓取 Yahoo Finance 的兩年期日線歷史數據，精確評估 7 大趨勢樣板過濾指標、計算波動收縮波段（C1-C4），並自動輸出高對比、資訊極其清晰且防守紀律分明的 HTML 互動診斷報告。

---

## 🚀 核心功能特色

1. **無外部相依性（Zero Dependencies）**：完全採用 Node.js 原生 `fetch` 抓取 Yahoo Finance API 歷史價格數據，無須安裝任何繁重的 npm 套件，運行極速且極易部署。
2. **雙模式靈活運行**：
   * **個股深度技術診斷**：對指定股票進行精確的均線、價格趨勢、VCP 收縮波及部位風控規劃。
   * **批次清單選股篩選**：讀取自訂的個股清單檔案，自動批次篩選並生成選股儀表板，一目了然所有股票的多頭評分。
3. **按年月自動歸檔**：生成的個股報告與儀表板會自動依當前年月命名並建立子資料夾（如 `reports/2026-05/`）集中存放，徹底告別混亂的根目錄。
4. **資訊極致清晰的 TradingView 風格報表**：
   * **過濾指標透明拆解**：7 大均線與動能指標逐條列出「判定公式」、「實際數據」、「判定結果（PASS/FAIL）」與「籌碼學過濾合理解釋」。
   * **波動收縮歷程表**：清楚列出 C1 至 C4 的收縮階段，標註每個波段的最高價、最低價、修正幅度和天數。
   * **原生 SVG 圖表**：免除外部圖表庫負擔，直接以 SVG 繪製最近 100 天的實際價格曲線與 50MA, 150MA, 200MA。
   * **互動式部位規模風險計算器**：內建原生 JS 計算器。使用者填寫「帳戶總資金」與「單筆承受風險百分比（如 1%）」，即可結合最新股價與精確止損位，自動算出應買入股數與配置權重，極具實操價值。

## 💻 運行控制指令

請先開啟終端機（如 PowerShell 或 cmd），並確認工作目錄位於專案根目錄下：

### 1. 單股深度技術診斷模式
若要針對**某一檔特定股票**進行完整的 VCP 型態診斷與量價分析，請在代碼後附帶股票代號：
```powershell
node sepa_screener.js <TICKER>

# 執行範例：
node sepa_screener.js NVDA
node sepa_screener.js GOOG
node sepa_screener.js TSLA
```
* **產出報告路徑**：`reports/YYYY-MM/<ticker>_vcp_analysis.html`

### 2. 批次清單選股篩選模式
若要同時對**多檔個股**進行大盤點與均線樣板篩選：
1. 修改根目錄下的 `tickers.json` 檔案，填入您想監控的所有美股代碼，例如：
   ```json
   [
     "NVDA",
     "AAPL",
     "MSFT",
     "GOOG",
     "TSLA",
     "META",
     "AVGO",
     "NFLX",
     "AMD",
     "UUUU"
   ]
   ```
2. 在終端機直接執行無參數指令：
   ```powershell
   node sepa_screener.js
   ```
* **產出報告路徑**：
  * 系統會自動為清單中每一隻有數據的個股在年月資料夾下生成獨立的 HTML 診斷報告。
  * 並在該年月資料夾下生成彙整儀表板：`reports/YYYY-MM/sepa_screener_result.html`（可直接點選各股連結跳轉至對應報告）。

## 📈 馬克．米奈爾維尼 7 大趨勢樣板過濾標準

系統在分析時，會無條件對個股進行以下 7 項技術指標校驗，只有通過的股票才被視為健康的 **Stage 2（第二階段主升段）** 多頭個股：

1. **股價高於 150MA 且高於 200MA** —— 中長期趨勢向上，已脫離底部套牢盤。
2. **150MA 高於 200MA** —— 多頭排列確立，中期成本高於長期成本。
3. **200MA 呈現上升趨勢至少一個月** —— 長線生命線具有穩健的正斜率，長線大資金進場支撐。
4. **50MA 高於 150MA 且高於 200MA** —— 均線呈扇形排列，短中長成本依序墊高。
5. **股價高於 50MA** —— 短線動能強勁，回踩均線不破。
6. **股價比 52 週最低點上漲至少 30%** —— 確認累積足夠反彈動能，擺脫盤整築底期。
7. **股價距離 52 週最高點在 25% 以內** —— 鄰近歷史高點，上方套牢盤極薄，易於爆發突破。

## 🛡️ 交易防守與紀律提示

馬克．米奈爾維尼強調：「控制虧損是業餘與專業交易員的最大分水嶺。」

* **嚴格止損硬限制**：單筆交易的硬止損位必須設定於進場成本的 **4% ~ 8%** 之間，且在任何情況下**絕對不能超過 10%**。如果一隻股票需要大於 10% 的防禦空間，說明進場點選擇錯誤。
* **技術防守失效點**：本系統會自動將止損位設在**最近一次波動收縮（如 C3 或 C4）低點的下方 1% 處**（並確保符合 4%~8% 的風控限制）。一旦股價以大陰線伴隨放量跌破此防守點，代表突破失敗，應立即執行紀律停損。
* **部位配置原則**：單筆持股的最大部位不超過帳戶總資金的 **25%**。當單筆部位止損設為 5% 時，單筆交易對總帳戶的曝險僅為 $25\% \times 5\% = 1.25\%$，安全完全在可控範圍內。
* **漸進式暴露（Progressive Exposure）**：先建立第一筆試探部位，當該部位脫離成本區且產生帳面獲利時，才允許在後續的健康回踩中進行加碼，切忌一次性滿倉梭哈。
